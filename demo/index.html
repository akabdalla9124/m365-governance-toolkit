What This Toolkit Does

A PowerShell-based admin toolkit for governing Microsoft 365 environments — SharePoint Online, Teams, OneDrive, and Exchange. Built from real operational experience managing multi-site M365 deployments.

ScriptCategoryWhat It Does
New-UserOnboarding.ps1ProvisioningCreates user, assigns license, adds to SharePoint and Teams
Remove-UserOffboarding.ps1LifecycleRevokes access, transfers OneDrive, archives mailbox, removes licenses
Get-SiteGovernanceReport.ps1AuditingFlags sites missing owners, with external sharing, or inactive
Get-AccessReview.ps1AuditingExports all permissions and flags external/guest users
New-SharePointSiteProvisioning.ps1ProvisioningCreates governed sites from a standard template
Set-TeamsLifecyclePolicy.ps1GovernanceFlags inactive, ownerless, or policy-violating teams

New-UserOnboarding.ps1

Provisions a new M365 user end-to-end: account creation, license assignment, SharePoint group membership, and Teams access — in a single command.

PowerShell — New-UserOnboarding.ps1
.\scripts\New-UserOnboarding.ps1 ` -FirstName "Sarah" -LastName "Chen" -Department "Finance" ` -Manager "j.miller@contoso.com" -LicenseSku "E3" ` -SharePointSiteUrl "https://contoso.sharepoint.com/sites/finance" ` -TeamsTeamName "FIN-Budget Planning 2025" [1/5] Connecting to Microsoft Graph, SharePoint, Teams... Remove-UserOffboarding.ps1

Clean, auditable offboarding in one command: revokes sessions, removes from all groups, transfers OneDrive, converts mailbox to shared, removes licenses, disables account. Every action is written to a timestamped audit log.

PowerShell — Remove-UserOffboarding.ps1
.\scripts\Remove-UserOffboarding.ps1 ` -UserPrincipalName "m.williams@contoso.com" ` -OneDriveTransferTo "j.miller@contoso.com" ` -ArchiveMailbox $true [CONNECT] Connecting to Microsoft 365 services... [OK] Connected. [VALIDATE] Target user: Marcus Williams (m.williams@contoso.com) [SESSIONS] Revoking all active sign-in sessions... [OK] All sessions revoked. User is now signed out of all devices. [GROUPS] Removing from Microsoft 365 groups and Teams... [OK] Removed from 4 groups: OPS-Logistics Q1, Operations Members, All Staff, FIN-Budget Planning 2025. [ONEDRIVE] Transferring OneDrive to j.miller@contoso.com... [OK] OneDrive ownership transferred: /sites/m_williams_contoso_com/ [MAILBOX] Converting mailbox to shared mailbox... [OK] Mailbox converted. Full access granted to j.miller@contoso.com. [LICENSES] Removing Microsoft 365 E3 license... [OK] 1 license removed. [DISABLE] Disabling account... [OK] Account disabled. Offboarding complete for Marcus Williams. Audit log saved to: .\logs\offboarding-2026-04-22-1738.log
